Gitblogに投稿するまでの手順メモ

Nov 29th, 2013

新しい記事を作成

rake new_post[‘post title’]

※ここでのタイトルに日本語は使えない。また、ページのURLにもなる。
source/_post/YYYY-MM-DD-title.markdownというデータが出来上がるので、
中編集して記事を作成。

プレビュー

rake preview

http://localhost:4000/にアクセスすると見れる。Ctrl+cで終了。

記事のアップ

rake generate rake deploy

または上記２つを１つに

rake gen_deploy

ソースをBitbucketにバックアップ

git add -A git commit “add post” git push -u bitbucket source
