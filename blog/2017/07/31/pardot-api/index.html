バージョン

APIのバージョンは、3と4が稼働中のよう。
使い分けは以下との事。

同じメールアドレスで複数のプロスペクトを作成できないアカウントの場合、バージョン 3 を参照してください。
同じメールアドレスで複数のプロスペクトを作成できるアカウントの場合、バージョン 4 を参照してください。
