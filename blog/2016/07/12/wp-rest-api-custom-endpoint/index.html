メソッドの指定方法

メソッドの指定方法はGETやPOSTなど文字列での指定も可能だが、WP_REST_Serverクラスを指定して複数いっぺんに指定が出来たりもする。

  • READABLE = ‘GET’
  • CREATABLE = ‘POST’
  • EDITABLE = ‘POST, PUT, PATCH’
  • DELETABLE = ‘DELETE’
  • ALLMETHODS = ‘GET, POST, PUT, PATCH, DELETE’
